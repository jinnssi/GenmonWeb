# 🕶EYE WARE shopping mall🕶
Java, Oracle, JSP 를 사용하여 쇼핑몰 홈페이지를 제작한 프로젝트입니다.
# 
# 개발환경
* OS: Windows 10
* Language: Java 1.8.0
* DBMS: Oracle 18c
* Framework: Bootstrap
* Server: Tomcat 9.0
* Front-End: HTML, CSS, javascript, jQuery
# 
* 김지은
  - 문의 등록 (회원/비회원), 문의 목록, 관리자 – 문의 관리, 상품 등록 및 관리
